🔐 Firebase Configuration
Enter your Firebase project credentials to enable authentication
📖 How to get Firebase credentials:
  1. Go to Firebase Console
  2. Create a new project or select existing
  3. Click the gear icon (⚙️) → Project settings
  4. Scroll to "Your apps" section
  5. Click Web app (</>) if not created
  6. Copy the firebaseConfig values
  7. Enable Authentication → Sign-in methods → Email/Password & Phone

📧 EmailJS Configuration
Enter your EmailJS credentials to enable email features
📖 How to get EmailJS credentials:
  1. Go to emailjs.com and log in
  2. Public Key: Account → API Keys → Public Key
  3. Service ID: Email Services → your service → Service ID
  4. Template ID: Email Templates → your template → Template ID
